Pokedex!

This app is a quick reference for facts and images of all your favorite pokemon. Click any pokemon name button to see a picture of the pokemon, as well as height, weight, and type.

This app was built using HTML5, SCSS, and JavaScript ES6. After learning to create modals from scratch, I used bootstrap frameworks to create the navbar and modals. Pokemon data is drawn from the external api, https://pokeapi.co/, using AJAX. I used ESLint and Prettier to ensure my code was error-free.
